# Миграция оружия, боеприпасов и снаряжения

Запускайте импорт в порядке: **материалы → боеприпасы → оружие → снаряжение**.

- `00-ONE-MACRO-IMPORT-ALL-GEAR.js` — один макрос на всё
- `00-repair-gear-crafts.js` — **только крафты** (если на вкладке «Крафт» пусто кроме корневого предмета)
- или по отдельности из папок `ammo/`, `weapons/`, `equipment/`

В сгенерированных макросах ~1600 предметов с ингредиентами (778 оружие + 826 снаряжение). Если после импорта рецепты пустые — импорт прервался до второго прохода; запустите `00-repair-gear-crafts.js`.
